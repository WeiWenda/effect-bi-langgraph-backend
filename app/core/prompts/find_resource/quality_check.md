# 评判标准（仅以下三项，其它一律不考虑）

1. **与用户原始问题是否匹配**：检索到的表/元数据是否与用户原始问题所描述的对象、主题、意图一致。
2. **业务域是否匹配**：检索结果中的业务域是否与预处理结果中明确的**业务域**一致；预处理为「不限」时，检索结果应覆盖用户问题相关的域，且不应明显偏域。
3. **数仓分层是否匹配**：检索结果中的数仓分层是否与预处理结果中明确的**数仓分层**一致；预处理为「不限」时，检索结果的分层应合理覆盖用户问题所需层级，且不应明显错位。

以下情况判为**不达标（N）**：
- 检索结果为空或与用户原始问题无关
- 检索表的 `domain` / 业务域与预处理指定的业务域不一致（且预处理非「不限」）
- 检索表的 `layer` / 数仓分层与预处理指定的分层不一致（且预处理非「不限」）

以下情况**不要**作为评判依据：字段是否齐全、是否有备选方案、表述是否美观、不确定性说明等。

# 输出要求（严格遵守，仅输出一行）

若三项均满足，仅输出：`Y`  
否则仅输出：`N`  

不要输出其他任何文字。
