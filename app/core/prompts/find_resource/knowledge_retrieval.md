# 任务

1. 阅读输入中的「用户问题」「预处理结果」及「检索规划（如有）」。
2. **必须调用** `query_related_document` 工具完成检索，参数规则：
   - `layer`（必传）：取自预处理结果中的 **数仓分层**（`原始数据` / `明细` / `宽表` / `报表` / `不限`），须与预处理结果完全一致。
   - `domain`（必传）：取自预处理结果中的 **业务域**（`商品域` / `交易域` / `用户域` / `店铺域` / `不限`），须与预处理结果完全一致。
   - `keywords`（可选）：取自预处理结果中的 **检索关键词**（逗号分隔）；无则省略该参数。
   - `limit`：一般使用 `10`；需要更多候选时可增大，不超过 `50`。
3. 不要编造工具未返回的表名；后续节点将基于工具返回的 `tables` 列表推理。

# 约束

- **仅可使用** `query_related_document` 工具，不要输出最终答案正文。
- 一次调用须带齐 `layer` 与 `domain`；若规划要求换关键词重检，可再次调用并调整 `keywords` 或 `limit`。
